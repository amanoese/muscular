# Muscular !!!

Muscularは筋肉の力は最高のパワーであることを表現するためのコマンドです。

## Install

```npm install
$ npm i -g @amanoese/muscular
```

## Usage

muscular shoutは、ボディビルの掛け声とAAを表示してくれます。
また、AAと掛け声はランダムに変わります。掛け声を指定することもできます。
